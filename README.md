# 💌 HeartLine Bot

Telegram-бот для автоматичних персональних повідомлень близьким людям.

## Що вміє бот

- Додавати одержувачів (мама, тато, дружина, брат, друг, бабуся)
- Надсилати повідомлення за розкладом автоматично
- Генерувати текст за тоном (тепло, весело, романтично, спокійно)
- Додавати погоду та місто у повідомлення
- Три режими: авто / показати перед відправкою / ручне редагування
- Тестове надсилання в будь-який момент

## Швидкий запуск на Railway (безкоштовно)

### Крок 1 — Завантажити код на GitHub

1. Зареєструйся на https://github.com
2. Створи новий репозиторій (New repository) → назви `heartline-bot`
3. Завантаж всі файли з цієї папки

### Крок 2 — Запустити на Railway

1. Зайди на https://railway.app
2. Натисни "New Project" → "Deploy from GitHub repo"
3. Обери репозиторій `heartlinebot`
4. Перейди в Settings → Variables і додай:
   BOT_TOKEN = твій токен від BotFather
5. Railway автоматично запустить бота

### Крок 3 — Перевірити

Відкрий Telegram, знайди свого бота і напиши /start

## Додаткові можливості (опціонально)

### AI-генерація тексту (OpenAI)
1. Зареєструйся на https://platform.openai.com
2. Створи API ключ
3. Додай в Railway Variables: OPENAI_API_KEY = sk-...

### Реальна погода (Weatherstack)
1. Зареєструйся на https://weatherstack.com (є безкоштовний план)
2. Скопіюй API ключ
3. Додай в Railway Variables: WEATHER_API_KEY = ...

## Структура файлів

- bot.py — головний файл бота, всі команди та меню
- database.py — база даних SQLite (користувачі, одержувачі)
- ai_generator.py — генерація тексту (OpenAI або шаблони)
- weather.py — отримання погоди
- scheduler.py — планувальник автоматичних відправок
- requirements.txt — залежності Python
- railway.toml — конфігурація деплою
